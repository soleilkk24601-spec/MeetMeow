## 🌟 项目概述

**MeetMeow** 是一款基于 AI 视觉识别技术的城市猫咪互动记录微信小程序。我们致力于通过技术手段，为每一只在街角偶遇的流浪猫建立独特的身份档案，将单纯的“遇见”转化为有温度的“故事”与有力的“救助”。

- **对个体用户**：它是你的口袋猫咪图鉴，记录每一次邂逅。
- **对救助组织**：它是精准的协同工具，通过 LBS 地图实现高效救助。

---

## 🚀 核心价值

### 📸 智慧识咪：从镜头到故事
利用 AI 视觉识别，不仅能识别猫咪品种，还能深入分析其健康状况与情绪状态。通过 AI 共创技术，你可以与偶遇的猫咪“对话”，生成专属的故事卡片。

### 📚 收藏图鉴：建立城市猫咪档案
每一只你遇见的猫咪都会被收录在个人图鉴中。随着脚步的延伸，你将拥有一部专属于自己的城市猫咪百科全书。

### 💬 智能助手：全天候养护顾问
内置全局智能助手，无论是猫咪护理知识还是紧急救助指南，点击悬浮球即可随时咨询。

### 🌐 救助协同：LBS 赋能内容广场
基于地理位置信息的社交广场。普通用户分享温暖瞬间，认证的动保组织（NGO）则可以通过精确的经纬度定位和地图导航，快速响应救助需求。

## 🛠️ 功能模块

| 模块 | 功能描述 |
| :--- | :--- |
| **📷 AI 拍照识别** | 实时拍摄或上传图片，获取猫咪品种、健康、情绪及环境的多维度分析。 |
| **✍️ 互动共创** | 支持语音或文字输入，AI 会根据识别结果与用户共同谱写温情的猫咪故事。 |
| **📖 喵喵图鉴** | 结构化展示已收藏的猫咪记录，支持按情绪标签查看。 |
| **🌍 内容广场** | 朋友圈式的信息流，支持点赞、分享，沉淀城市猫咪的足迹。 |
| **📍 救助地图** | 针对 NGO 账号提供精准定位与导航功能，优化 TNR（抓捕、绝育、放归）流程。 |
| **🛡️ 组织认证** | 完善的 NGO 认证体系，区分普通用户与救助者权限，保护猫咪隐私。 |

## 👥 用户角色

> **MeetMeow 针对不同群体设计了差异化的功能体验：**

* **普通猫友 (User)**
    * 分享猫咪趣闻，收藏偶遇瞬间。
    * 广场展示模糊位置，兼顾分享欲与猫咪安全。
* **动保组织 (NGO)**
    * 查看广场帖子的精确 GPS 坐标。
    * 一键开启地图导航，直达救助现场。
    * 专业的身份标识，建立救助公信力。
